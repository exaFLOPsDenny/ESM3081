## Linear Models

### Linear Regression
$$ \hat{y} = w^\top x + b $$
$$ when \ \ x_i = \{ 1,\ x_{i1},\ x_{i2},... \} $$
$$ \hat{y} = w^\top x $$

> Finds the parameters **w** and b that minimize the MSE between predictions and true target values
> Has no hyperparameters, thus no way to control model complexity
